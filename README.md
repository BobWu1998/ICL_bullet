# ICL_bullet
## Installation
```
conda create --name bullet
conda activate bullet
pip install gym==0.18.0
pip3 install pybullet --upgrade --user
```
## to run sample envs
```
conda activate bullet_env
cd ICL_bullet
python -m kukaMulObj_test

```
python3 -m pybullet_envs.examples.kukaGymEnvTest2
```
